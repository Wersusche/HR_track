HR Track installer Условия использования

Политика конфиденциальности HR Track

Последнее обновление: 10 апреля 2026 года.

HR Track — некоммерческий инструмент для личного учета откликов на вакансии. Он помогает пользователю создать собственную Google Таблицу, связанный Apps Script web app и персональный userscript для Tampermonkey.

Какие данные используются

Приложение запрашивает доступ к Google API только для создания и настройки ресурсов в аккаунте пользователя: Google Таблицы, Apps Script проекта и deployment web app. Данные об откликах, заметках, компаниях, вакансиях и ссылках сохраняются в Google Таблице, принадлежащей пользователю.

Где хранятся данные

У HR Track нет собственного backend-сервера и собственной базы данных. OAuth access token используется в браузере во время настройки и не сохраняется на сервере HR Track. Конфигурация установки может сохраняться локально в браузере пользователя через localStorage.

Передача данных третьим лицам

HR Track не продает, не передает и не раскрывает пользовательские данные третьим лицам в коммерческих целях. Данные передаются только в Google API настолько, насколько это необходимо для работы выбранных пользователем функций.

Использование Google user data

Доступ к данным Google используется только для функций HR Track: создания таблицы, записи структуры таблицы, создания Apps Script проекта и публикации web app. Эти данные не используются для рекламы, аналитического профилирования или коммерческой перепродажи.

Контроль доступа

Пользователь может удалить созданные Google Таблицы и Apps Script проекты в своем Google аккаунте, удалить userscript в Tampermonkey и отозвать доступ приложения в настройках безопасности Google аккаунта.
